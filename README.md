# odin-recipes
This is my first project submission for the HTML Basics Module in The Odin Project Web development Course.

The task was to create a simple website only with HTML.

The website is for hosting recipes, it includes simple navigation between the home page and pages holding recipes for the relevant food item.

This project taught me the basics of created a static site which includes simple navigation with anchor tags, placing images on the screen, and adding content to the page
in a simple and clean format. I expect to add some style to this static site in future.
